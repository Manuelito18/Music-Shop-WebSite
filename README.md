# music-shop-PHP

Este proyecto es una aplicación web desarrollada con **React** y **Vite** para el frontend, y **PHP** como backend.

## Tecnologías

- **Frontend:** React + Vite
- **Backend:** PHP

## Instalación

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
composer install
php -S localhost:8000
```

## Estructura del proyecto

```
music-shop-PHP/
├── frontend/   # React + Vite
└── backend/    # PHP
```
